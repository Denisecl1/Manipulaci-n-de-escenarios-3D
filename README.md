# 🎮 Galería de Demos Three.js

Proyecto web interactivo que presenta diferentes ejemplos de **Three.js** para explorar controles y entornos 3D directamente desde el navegador.

Este proyecto fue desarrollado como parte de la materia **Ambientes de Desarrollo Virtuales**, mostrando distintas formas de interactuar con escenas 3D utilizando JavaScript.

---

## 📌 Descripción

La aplicación consiste en una **galería de demos** que permite acceder a diferentes ejemplos desarrollados con **Three.js**, cada uno mostrando distintos tipos de controles y comportamientos en un entorno 3D.

El objetivo es **explorar las capacidades de renderizado y navegación en 3D dentro del navegador** mediante tecnologías web modernas.

---

## 🖼️ Interfaz principal

La página principal presenta una **galería de demos interactivos** donde el usuario puede seleccionar qué ejemplo visualizar.

Características de la interfaz:

* Navbar de navegación
* Galería de demos
* Botones interactivos
* Footer informativo
* Diseño responsive con Bootstrap

---

## 🚀 Demos incluidos

### 🧱 Minecraft Demo

Simulación de un entorno estilo **Minecraft** donde el usuario puede moverse dentro de un mundo compuesto por cubos.

**Controles**

* `W A S D` → mover jugador
* `SPACE` → saltar
* `Mouse` → mirar alrededor

---

### 🗺️ Map Controls

Demostración de controles tipo **mapa**, utilizados comúnmente en visualizadores 3D o mapas interactivos.

Permite:

* Rotar la cámara
* Desplazarse en el entorno
* Hacer zoom

---

### 🔄 Orbit Controls

Ejemplo de los controles **OrbitControls** de Three.js.

Permite:

* Rotar alrededor de un objeto
* Zoom con la rueda del mouse
* Desplazar la cámara

Este tipo de control se usa mucho en **modeladores 3D y visualizadores de objetos**.

---

### 🎯 Pointer Lock (First Person)

Simulación de movimiento en **primera persona**, similar a videojuegos FPS.

Características:

* Movimiento libre en el escenario
* Mira con el mouse
* Interacción con el entorno
